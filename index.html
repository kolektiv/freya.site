Freya

An Open Source Functional Web Stack for F#

Get Started Now

Open

Freya is a genuinely open source project. Development and the roadmap is done in public, pull requests are welcome, and we listen to anyone that wants to use Freya. We do everything we can to make Freya friendly.

Functional

Freya embraces the power of F# and a functional approach to web programming, allowing it to fit seamlessly with your existing functional code. Of course, it can happily work with OO code if you have some!

Stack

Freya is a stack not a framework. You can choose which parts of Freya you wish to use — or use everything (just pick the Freya meta package from NuGet). It's easy to extend and specialise Freya to be the perfect stack for you.
